## Summary

This public demo shows an AI investment-analysis prompt structure as deterministic local research artifacts, so the structure can be inspected instead of being trusted blindly.

- Research-only public demo.
- Sample-only artifacts.
- Public demo repo only.
- Private development repo remains private.
- Local CSV artifacts only.
- Deterministic outputs.
