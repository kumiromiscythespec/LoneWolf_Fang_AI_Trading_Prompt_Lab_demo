# Demo Artifact Map

This map explains the public `demo/` folder.

## Root Demo Files

| Path | Purpose | What It Does Not Prove |
| --- | --- | --- |
| `demo/demo_manifest.json` | Generated manifest, safety boundary, artifact list, schema check summary, and safety scan result. | It does not prove production readiness or market value. |
| `demo/demo_safe_summary.md` | Human-readable generated safety summary. | It does not provide financial advice. |
| `demo/demo_checksums.sha256` | SHA-256 list for generated files. | It does not certify private source content. |
| `demo/demo_index.html` | Browser index for the static demo. | It does not connect to live services. |
| `demo/demo_artifact_pack.zip` | Generated pack of selected demo artifacts. | It does not contain private source code. |

## Dashboard

| Path | Purpose | What It Does Not Prove |
| --- | --- | --- |
