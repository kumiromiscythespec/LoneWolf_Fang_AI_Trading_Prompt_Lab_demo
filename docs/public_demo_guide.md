# Public Demo Guide

This guide explains how to inspect the public demo without running code or connecting to any outside service.

## Scope

- Public demo repo only.
- Research-only.
- Sample-only.
- Local CSV demo artifacts only.
- Static files only.
- No financial advice.
- No order execution.
- No private API.
- No external AI.

## Opening The Demo

Open `demo/demo_index.html` directly in a browser. It links to the generated reports, summaries, dashboard, schema checks, manifest, and checksum list.

The demo does not require a server. It does not require an API key, account, wallet, broker, exchange connection, remote model, or cloud runtime.

## Dashboard

Open `demo/dashboard/dashboard.html` to view the static dashboard. The dashboard is a generated local artifact built from sample CSV inputs. It is not connected to live data and does not send requests to trading services.

The related files are:

- `demo/dashboard/dashboard_index.json`
- `demo/dashboard/dashboard.md`
- `demo/dashboard/dashboard.html`

## Summaries

The `demo/summaries/` folder contains compact generated summaries for public review:

- `analyze_summary`
- `compare_summary`
- `diff_summary`
- `dashboard_summary`

Each folder includes Markdown, JSON, and HTML versions where generated by the demo pack.

## Schema Checks

The `demo/schema_checks/` folder shows compatibility checks for generated JSON artifacts. A compatible result means the demo artifact matched the expected public schema shape. It does not prove trading value, future market behavior, account safety, or production readiness.

## Checksums

Use `demo/demo_checksums.sha256` to review generated file hashes. The artifact pack zip hash is also recorded in `demo/demo_manifest.json`.

The convenience copy at `demo/artifact_pack/demo_artifact_pack.zip` has a matching hash listed in `demo/artifact_pack/demo_artifact_pack.sha256.txt`.

## Sample-Only Limitation

The demo uses sample CSV data only. It is designed for reviewing artifact shape, documentation clarity, and safety boundaries. It is not a trading system and is not a financial advice product.
