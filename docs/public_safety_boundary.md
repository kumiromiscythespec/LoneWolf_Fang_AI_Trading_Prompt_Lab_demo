# Public Safety Boundary

This public demo is intentionally narrow.

## Allowed Scope

- Research-only inspection.
- Sample-only artifacts.
- Local CSV demo inputs only.
- Deterministic static outputs.
- Local HTML, Markdown, JSON, checksum, and zip artifacts.

## Explicit Non-Scope

- No trading.
- No order execution.
- No broker connection.
- No exchange private API.
- No account state.
- No wallet access.
- No API keys or secret keys.
- No external AI.
- No cloud summarization.
- No server runtime.
- No webhook trading.
- No auto trading.
- No financial advice.
- No profit claim.

## Private Repo Boundary

The private development repo remains private. This public repo excludes private source code, tests, full fixtures, development notes, private exports, and unrelated LoneWolf Fang files.

## Public Interpretation

The artifacts are for reviewing a deterministic local research workflow. They should be read as sample evidence of artifact generation, static documentation, schema checks, and checksum hygiene only.
